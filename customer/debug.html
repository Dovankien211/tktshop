📋 Các vấn đề phổ biến được phát hiện:

  • 🚨 Đường dẫn tuyệt đối sai: Một số file sử dụng /tktshop/ thay vì đường dẫn tương đối
  • ⚠️ Include/Require sai: Đường dẫn đến config.php và database.php không chính xác
  • 🔗 Link CSS/JS bị lỗi: Tham chiếu đến assets không tồn tại
  • 📂 Redirect không đúng: Các trang redirect sai thư mục

🚨 Các lỗi đường dẫn được phát hiện

❌ Lỗi nghiêm trọng - Config Path

customer/includes/header.php

Vấn đề: Đường dẫn đến config sai

// ❌ SAI require_once '/tktshop/config/config.php'; // ✅ ĐÚNG require_once '../config/config.php';

❌ Database Connection

customer/*.php

Vấn đề: Include database.php không đúng đường dẫn

// ❌ SAI include_once '/config/database.php'; // ✅ ĐÚNG require_once '../config/database.php';

⚠️ CSS/JS Assets

customer/includes/header.php

Vấn đề: Link CSS/JS không tồn tại

// ❌ SAI - không có thư mục assets <link rel="stylesheet" href="/assets/css/style.css"> // ✅ ĐÚNG - sử dụng CDN hoặc tạo thư mục assets <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.1.3/css/bootstrap.min.css">

⚠️ Admin Layout Issues

admin/layouts/header.php

Vấn đề: Include config từ admin folder

// ❌ SAI require_once 'config/config.php'; // ✅ ĐÚNG require_once '../../config/config.php';

ℹ️ Image Paths

customer/products.php

Vấn đề: Đường dẫn ảnh sản phẩm không nhất quán

// ❌ Không nhất quán <img src="/uploads/products/"> // ✅ ĐÚNG - sử dụng base URL <img src="/uploads/products/">

🔧 Các file cần sửa lỗi

📄 1. config/config.php - Cấu hình cơ bản

<?php // Cấu hình đường dẫn cơ bản define('BASE_PATH', dirname(__FILE__) . '/../'); define('BASE_URL', 'http://localhost/tktshop'); // Thay đổi theo domain của bạn define('ADMIN_URL', BASE_URL . '/admin'); define('CUSTOMER_URL', BASE_URL . '/customer'); define('UPLOAD_PATH', BASE_PATH . 'uploads/'); define('UPLOAD_URL', BASE_URL . '/uploads'); // Database config define('DB_HOST', 'localhost'); define('DB_USER', 'root'); define('DB_PASS', ''); define('DB_NAME', 'tktshop'); // Session config ini_set('session.cookie_httponly', 1); session_start(); ?>

📄 2. customer/includes/header.php - Header đã fix

<?php require_once '../config/config.php'; require_once '../config/database.php'; ?> <!DOCTYPE html> <html lang="vi"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title><?php echo isset($page_title) ? $page_title . ' - ' : ''; ?>TKTShop</title> <!-- Bootstrap CSS --> <link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.1.3/css/bootstrap.min.css" rel="stylesheet"> <!-- Font Awesome --> <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css" rel="stylesheet"> </head>

📄 3. admin/layouts/header.php - Admin header

<?php require_once '../../config/config.php'; require_once '../../config/database.php'; // Kiểm tra đăng nhập admin if (!isset($_SESSION['admin_id'])) { header('Location: ' . BASE_URL . '/admin/login.php'); exit; } ?>

📄 4. index.php (root) - Redirect chính xác

<?php // Redirect về trang customer header('Location: customer/index.php'); exit; ?>

📄 5. .htaccess - URL Rewrite & Security

RewriteEngine On # Redirect root to customer RewriteRule ^$ customer/ [R=301,L] # Security - Block access to config files <Files "*.php"> Order Allow,Deny Allow from all </Files> <Files "config.php"> Order Deny,Allow Deny from all </Files> # Prevent directory browsing Options -Indexes

⚙️ Cấu hình hệ thống

🔧 Các bước sửa lỗi theo thứ tự:

  1. Bước 1: Cập nhật config/config.php với BASE_URL đúng
  2. Bước 2: Sửa tất cả file includes trong customer/
  3. Bước 3: Sửa tất cả file includes trong admin/
  4. Bước 4: Kiểm tra đường dẫn ảnh trong DB
  5. Bước 5: Test từng trang một

📝 Script kiểm tra nhanh

Tạo file test_links.php để kiểm tra:

<?php echo "<h2>Kiểm tra đường dẫn</h2>"; // Test config if (file_exists('../config/config.php')) { echo "✅ Config file exists<br>"; require_once '../config/config.php'; echo "✅ BASE_URL: " . BASE_URL . "<br>"; } else { echo "❌ Config file not found<br>"; } // Test database if (file_exists('../config/database.php')) { echo "✅ Database file exists<br>"; } else { echo "❌ Database file not found<br>"; } // Test uploads folder if (is_dir('../uploads/')) { echo "✅ Uploads folder exists<br>"; } else { echo "❌ Uploads folder not found<br>"; } ?>

🎯 Lưu ý quan trọng:

  • BASE_URL: Thay đổi trong config.php theo domain thực tế của bạn
  • Đường dẫn tương đối: Luôn sử dụng ../ để đi lên thư mục cha
  • Database: Kiểm tra thông tin kết nối database
  • Permissions: Đảm bảo thư mục uploads có quyền ghi (755)
